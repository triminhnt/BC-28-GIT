1/ Khoi tao GIT
    - Lenh: git init

2/ Kiem tra phien ban GIT 
    - Lenh: git --version

3. Dua code cua minh len server gitdub/gitlap
    - Lenh: git add . | git add index.html
            git commit -m "your comment"
            git push origin master | git push
            git push --set-upstream origin master

4. Lenh kiem tra ra trong tang thai cua git 
    - Lenh git status

5. Lenh giup GIT biet duoc se dua code len cho nao
    - Lenh: git remote add {name} {url_github}
    Vi du: git remote add origin https://github.com/triminhnt/BC-28-GIT.git
    
6. Kiem tra xem add remote thanh cong chua
    - Lenh: git remote 

7. Nguoi moi vao cong ty lay code ve nhu the nao:
    - Lenh: git clone {url_github}
    - Lenh: git clone --branch {branch_name} {url_github}

    vi du: git clone https://github.com/triminhnt/BC-28-GIT.git
    vi du: git clone --branch master https://github.com/triminhnt/BC-28-GIT.git

8. Kiem tra xem co bao nhieu nhanh tren sourcecode:
    - Lenh: git branch

9. Tao them nhanh moi:
    - Lenh: git branch {new_branch}
    Vi du: git branch demo-branch

10. Chuyen nhanh:
    - Lenh: git checkout {branch_name}
    vi du: git checkout demo-branch
